Class: Form

ebaui.web. Form

new Form(options, element)

ebaui.web.Form 能批量对多个控件进行赋值、取值、重置、验证、获取错误信息等，节省大量针对单独控件的操作代码。

enctype :String

 规定表单数据在发送到服务器之前应该如何编码
 常用的值有：
 
 application/x-www-form-urlencoded
     在发送前编码所有字符（默认
 multipart/form-data 
     不对字符编码。
     在使用包含文件上传控件的表单时，必须使用该值。
 text/plain  
     空格转换为 "+" 加号，但不对特殊字符编码。

  • String
