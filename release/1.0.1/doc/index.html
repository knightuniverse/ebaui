开发环境搭建

项目构建

首先确保你的开发环境上已经安装好了以下环境：

  • java
  • nodejs

Grunt

然后我们要安装grunt以及coffeescript。打开命令窗口，运行以下代码：

npm install -g coffee-script && npm install -g grunt && npm install -g grunt-cli

接下来，我们就开始安装其他构建项目用的grunt任务了：

npm install

完成之后，你就已经把开发环境给搭建起来了。
